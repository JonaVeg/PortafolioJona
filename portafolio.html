Aplicacion web "Viaja Con Amigos"

Aplicación "Viaja con amigos" "Viaja con amigos" es una aplicación móvil destinada a los estudiantes de la Universidad Tecnológica de Puebla (UTP). La aplicación facilita el transporte compartido entre alumnos, permitiendo a aquellos que tienen coche, moto u otro medio de transporte compartir gastos y reducir la posibilidad de llegar tarde a clases. Además, la app fomenta la convivencia entre estudiantes durante los trayectos.

    Tecnologías utilizadas:
  • Dart y Flutter para el desarrollo de la aplicación móvil.
  • Firebase para la gestión de datos en la nube.
  • Android Studio como entorno de desarrollo.
Desarrollé una página web asociada a la aplicación, donde los usuarios pueden consultar los términos y condiciones, así como la política de privacidad. Puedes visitar la página aquí: Sitio web oficial. Lamentablemente, la aplicación nunca llegó a lanzarse en la Google Play Store debido a costos que no pude solventar en ese momento. A pesar de ello, la aplicación se mantiene como un prototipo, con la intención de lanzarla de manera funcional en el futuro, disponible para descarga.

Página Panther Shop

Página Panther Shop es una tienda en línea diseñada para la venta de artículos deportivos. Esta aplicación web fue desarrollada utilizando HTML, CSS y JavaScript, con el soporte del framework Bootstrap para ofrecer una interfaz atractiva y fácil de usar. Además, se implementó PHP y MySQL para manejar la lógica del servidor y la base de datos.

El objetivo principal de la página es proporcionar a los usuarios una plataforma en la que puedan explorar y adquirir productos deportivos, como playeras, shorts, calcetas, tenis, guantes, balones, y más, todo desde la comodidad de su hogar.

Página EVRepairs

EVRepairs se presenta como tu centro de confianza para la reparación de dispositivos móviles. El tipo de software es una aplicación web construida utilizando HTML, CSS y JavaScript, junto con el framework Bootstrap para mejorar el diseño y la experiencia del usuario. El propósito de la aplicación es ofrecer a los usuarios una plataforma en la que puedan explorar y destacar la sólida formación y experiencia técnica de su equipo de expertos, evidenciando un enfoque profesional y especializado en la reparación de dispositivos móviles.

Tecnologías: HTML, CSS, JavaScript, Bootstrap
